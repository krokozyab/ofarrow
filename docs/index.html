Why choose ofarrow?

Spin-up in seconds

Place ofarrow.jar next to any project, point it at an existing OTBI report and run: your Arrow Flight and REST endpoints are online in under a minute.

Format freedom

Give analysts Excel, data-engineers Parquet, developers JSON, and legacy systems CSV – all from a single URL parameter.

Cloud-scale speed

The service streams columnar Arrow batches that glide from Fusion into notebooks, dashboards or lake-houses with minimal overhead.

Zero lock-in

ofarrow is 100 % open source – no subscriptions, proprietary storage, or hidden throttles. It scales as your data lake grows.

Typical workflow

  1. Run java -jar ofarrow.jar.
  2. Explore data quickly in DBeaver, a Jupyter notebook, or any BI tool using the Arrow Flight connector.
  3. Export partitioned Parquet to S3 or ADLS for Athena, Starburst, or Snowflake.
  4. Automate daily jobs with simple curl calls – no heavyweight integration middleware.
