Why choose ofarrow?

Native installers

No Java, no scripts. Just install on macOS or Windows and start streaming Fusion data immediately.

One tool, any format

Give analysts Excel, data engineers Parquet, developers JSON, and legacy systems CSV – all with one server.

High-speed streaming

Arrow columnar batches move seamlessly into Pandas, Spark, or BI dashboards.

Open & vendor-neutral

No lock-in, no subscriptions. 100% open-source – scales with your data lake.

Quick start

  1. Install the macOS `.pkg` or Windows `.msi` from the releases page.
  2. Launch the app – your Arrow Flight and REST endpoints are up.
  3. Connect via Excel, Python, or Spark.
  4. Export data instantly in CSV, JSON, Excel, or Parquet.
